# 俄罗斯方块 🧱

> 经典俄罗斯方块在线游戏

**在线玩**: https://schaezel.github.io/tetris-russia/
**品牌**: HippieGames

## 游戏操作

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 直接落到底 |

## 计分规则

| 消除行数 | 得分 |
|---------|------|
| 1 行 | 100 分 |
| 2 行 | 300 分 |
| 3 行 | 500 分 |
| 4 行（Tetris） | 800 分 |

每消除 10 行升一级，速度加快。

## 功能

- ✅ 下一个方块预览
- ✅ 幽灵方块（显示落点）
- ✅ 最高分存档（localStorage）
- ✅ 等级加速系统
- ✅ 经典 7 种方块

## 开发

直接在浏览器打开 `index.html`，无需服务器。

```bash
# 本地预览
npx serve .
```
